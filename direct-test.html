Testare Pagini

Instrucțiuni pentru testare

  1. Apăsați pe unul din butoanele de mai sus pentru a deschide pagina dorită în browser.
  2. După ce pagina s-a deschis, deschideți consola browser-ului (F12 sau click dreapta -> Inspect -> Console).
  3. Copiați și lipiți următorul cod în consolă pentru a rula testele:
    4. // Pentru testul general
const script = document.createElement('script');
script.src = 'http://localhost:5173/direct-test.js';
document.head.appendChild(script);

// SAU pentru testul automat
const autoScript = document.createElement('script');
autoScript.src = 'http://localhost:5173/auto-test.js';
document.head.appendChild(autoScript);

// SAU pentru testul de interacțiuni
const interactionScript = document.createElement('script');
interactionScript.src = 'http://localhost:5173/interaction-test.js';
document.head.appendChild(interactionScript);
  4. Testele vor rula automat și rezultatele vor fi afișate în consolă.
  5. Verificați rezultatele pentru a vedea dacă pagina răspunde corect și nu are erori.
